Twilio Flex Plugin Builder

Monorepo for all the tooling needed to create a custom Twilio Flex plugins

Getting Started

This is a monorepo project managed by lerna for creating Twilio Flex plugins.

  • create-flex-plugin: The CLI tool to start a new Flex plugin project
  • flex-plugin: Runtime dependency for Flex plugins
  • flex-plugin-scripts: The scripts used for building, testing, and deploying your plugin
